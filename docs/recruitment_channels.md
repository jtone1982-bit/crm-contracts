# Рекрутмент-каналы и группы для рекламы службы по контракту

## Инструкция по сбору

### Telegram
1. Открыть https://tgstat.ru (с обычного браузера, без VPN/бота).
2. В поиск ввести запрос из столбца «Поисковый запрос».
3. Отсортировать по подписчикам.
4. Скопировать ссылку вида `https://t.me/username` и число подписчиков.
5. Проверить, что канал активен и публикует вакансии/контрактную тематику.

### VK
1. Открыть https://vk.com/groups.
2. В поиск ввести запрос из столбца «Поисковый запрос».
3. В карточке группы видно число подписчиков.
4. Скопировать ссылку и число подписчиков.

## Важно
- Цифры подписчиков быстро устаревают — уточнять перед покупкой.
- Рекрутмент-каналы часто закрыты или полуофициальные — связываться с админом.
- Для федерального охвата лучше комбинация: 2–3 крупных новостных канала + 10–15 региональных групп.
